HTML Tags

  1. html :

    Description: Defines the root element of an HTML document.

  2. head:

    Description: Contains meta-information about the document, such as the title and links to external resources.

  3. title:

    Description: Sets the title of the HTML document, which appears in the browser's title bar or tab.

  4. link:

    Description: Specifies external resources like stylesheets (CSS) or icons.

  5. style:

    Description: Contains CSS rules for styling the document.

  6. script:

    Description: Embeds or references JavaScript code for client-side scripting

  7. body:

    Description: Contains the visible content of the web page.

  8. header:

    Description: Represents introductory content or a group of navigation links at the top of a page or section.

  9. nav:

    Description: Defines a navigation menu, typically for site navigation.
